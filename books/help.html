Books

Help

With Books options you are be able to manage book list, draw a book and know who has a certain book.

  To look for a book:

  • Select the option Search a book.
  • Fill in the menu and push "Search book(s)". Remember the query is an and among the voices inserted in the menu and the search occurs matching any information having the inserted strings as an understring.
  • Disp="t" if the book is inserted, Disp="f" if the book is not.
  • Coll. gives the collocation of the book.
  To draw a book:
  • Select the option Draw a book.
  • Fill in the menu in all its parts fingering name and surname of user carefully, otherwise it signals Insert a new user (From 1st Nov 1996 onwards he/she must be inserted if it is the first time he/she draw a book or a review).
  • Insert the password="********".
  • Push "Draw book".
  • Control Disp="f" with the option Search a book.
  To deposit a book:
  • Select the option Deposit a book.
  • Fill in the menu in all its parts.
  • Insert tye password="********".
  • Push "Deposit book".
  • Control Disp="t" with the option Search a book.
  To insert a new user:
  • Select the link Insert a new user.
  • Fill in the menu in all its parts fingering name and surname carefully (It is advisable to put the initials in capital letters and the others in small letters).
  • Insert the password="********".
  • Push "Insert book".
  • Control if the insertion with the option User list order by surname has occurred.
  To know who has a certain book:
  • Select the option Who has a certain book ?.
  • Now, by the visualized menu, you can choose if you will have the list of the borrowed books or information about the user of a certain book.
  To know the place of books:
  • Select the option Book position.
  • Four images will be visualized: the 1st is the map of the library, the 2nd visualizes the shelves of the locker in the corridor, the 3rd the shelves in the eastern wall and the 4th the shelves in the wester wall.
